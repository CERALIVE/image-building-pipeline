# rk3588-userspace-deb-versions.txt — pinned RK3588 HW-accel userspace .debs
#
# WHAT: the Rockchip RK3588 GPU/video userspace (Mali-G610 blob, MPP encode/decode
# library, RGA 2D accelerator, the GStreamer MPP plugin, and the multimedia udev
# config) that the Armbian bookworm arm64 feed does NOT ship. These make the vendor
# kernel's HW encoders reachable from GStreamer (mpph264enc/mpph265enc/mppjpegenc/
# mppvp8enc register only when this MPP userspace is present) and give Cog/WPEWebKit
# a working Mali EGL/GBM stack for the 5.4 Kiosk render path.
#
# WHY PINNED URLs (not an apt source): these are NOT in a rotating apt repo we
# trust, so they are pinned to exact upstream release-asset URLs and verified by
# SHA-256 before staging — the SAME fail-closed, no-fallback discipline as the
# Armbian BSP fetch (v2/lib/fetch-debs.sh fetch_bsp), minus the apt/GPG machinery
# (a pinned URL + SHA-256 needs no rotating index or GPG trust root). Do NOT turn
# any of these into a `deb [signed-by=...] https://...` apt line — that is a new
# live trust root and is exactly what this file exists to avoid.
#
# PROVENANCE: proven working on real Rock 5B+ hardware (raw H.264/H.265 HW encode
# via mpph264enc/mpph265enc, ffprobe-verified). See
# .omo/notepads/production-launch-readiness/rk3588-userspace-packaging.md for the
# exact hardware-proof context and the source/variant decisions.
#
# FORMAT: one record per package, four whitespace-separated fields:
#   <package>  <filename>  <sha256>  <url>
#   * <package>  MUST equal the Debian Package: field — it is the family-manifest
#                wiring key AND the orchestrate.sh partition key.
#   * <filename> is the exact release-asset filename (== basename of <url>).
#   * <sha256>   is the real lowercase-hex SHA-256 of the exact asset.
#   * <url>      is the exact pinned download URL.
# Consumed by fetch_rk3588_userspace in v2/lib/fetch-debs.sh. The set of <package>
# names here MUST match the userspace packages declared in the rk3588 family
# manifest (firmware_packages / hw_accel_gstreamer_plugins / gstreamer_runtime_packages):
# fetch_bsp EXCLUDES these names from the Armbian fetch, and fetch_rk3588_userspace
# fetches exactly the ones the resolved family declares.
#
# VERSIONS ARE EMPIRICALLY PROVEN — do not bump without re-proving on hardware.
